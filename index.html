Контракты — экономика маржа из ABC-заказов · остаток из дебиторки на 16.06 · ₽

ЗаказчикРейсовМаржаОстаток к получениюСтатус
Как читать: «Маржа» — что компания заработала на заказе (из ABC). «Остаток» — сколько ещё должны (дебиторка). Выручка контракта ≈ остаток + уже оплачено; траты на перевозчика ≈ выручка − маржа. Оговорки: по НК НПЗ 1,28 млн — факторинг, деньги, возможно, уже получены от фактора (уточнить). По КРП остаток спорный (683к по ABC vs 1,1 млн в дебиторке). Где «тс едет/грузится» — рейс ещё не закрыт, это будущий доход. Точная экономика «сумма контракта / аванс / траты» будет считаться автоматически, когда операции начнут тегать проектом (лист «ПРОЕКТЫ»).
Методология: точка отсчёта — 01.06.2026, дальше текучка ведётся отдельно. Кэш-метод; переводы между своими счетами исключены; тело полученных кредитов доходом не считается. Остатки ООО на 01.06 сверены ПланФакт = УУ до рубля. По ИП/личному ПланФакт ненадёжен (дыры в истории), опираемся на выписки. Уточнено Эльвирой (18.06): остаток ИП 1376 на 01.06 = 0 + налички нет (выписка) · выручка с личных карт Озон/Т-дебет = доход ИП · оборотный кредит ООО = 3,27 млн (475 580/мес до 03.2027).
